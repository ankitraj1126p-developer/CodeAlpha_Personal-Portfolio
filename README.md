
 💼 Personal Portfolio Website

 📌 Project Overview

This is a **Personal Portfolio Website** built using HTML, CSS, and JavaScript.  
It showcases my skills, projects, and contact information in a clean and responsive layout.

The website is designed to be user-friendly and responsive, making it accessible on both desktop and mobile devices.

✨ Features

👤 About Me Section  

Displays personal introduction, background, and career goals.

🛠️ Skills Section  

Shows technical skills such as HTML, CSS, JavaScript, SQL, and GitHub.

📂 Projects Section  

Includes projects with **Live Demo** and **GitHub links**.

📱 Responsive Design  

Works properly on desktop, tablet, and mobile screens.

📧 Contact Section  

Provides email, phone, LinkedIn, and GitHub details.

📄 Resume Download  

Users can download resume directly from the website.

 🛠️ Technologies Used

HTML5 – Website structure  
CSS3 – Styling and layout  
JavaScript – Interactive features  
Font Awesome – Icons
